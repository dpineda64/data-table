# Props

## pages
- Type: `number[]`
- Required: `true`
- Description: _an Array with all the pages available_

## active
- Type: `number`
- Required: `true`
- Description: _current active page_

## nextPage
- Type: `function`
- Required: `true`
- Description: _handler for next page button_

## prevPage
- Type: `function`
- Required: `true`
- Description: _handler for prev page button_
